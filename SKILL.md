# Oliodo Copywriter Skill

## 🎯 スキルの概要

このスキルは、焼き菓子・洋菓子ブランド「Oliodo（オリオド）」の **全マーケティングコンテンツ** で一貫性のある、感情的で洗練されたコピーを生成します。

**Oliodoの本質：** お菓子そのものではなく、「季節の物語」「人生の節目」「日常の彩り」を顧客に届けるブランド。

---

## 📖 Oliodoの3つの哲学（必ず反映させること）

1. **季節の素材の活用**
   - 旬の素材を通じて、時間と記憶に寄り添う
   - コピーに「秋」「春」「新緑」など季節イメージを自然に織り交ぜる

2. **クリエイター・職人としての探究**
   - 「なぜこの素材なのか」「どうこだわったのか」を丁寧に表現
   - 職人目線より「顧客がいかに感動するか」を優先

3. **関係者への感謝**
   - 素材の生産者、顧客、仲間へのリスペクトを言葉に込める
   - 「作り手の想い」が顧客に伝わるようなストーリーテリング

---

## 🎨 ブランドボイス（トーン＆マナー）

### 基本原則
- **上品 × 親しみやすさ** — 詩的だが、押しつけがましくない
- **感情優先** — データより、口に含んだ瞬間のイメージ、記憶
- **季節感** — あらゆるコピーに時間軸・季節性を意識
- **「ご褒美」「節目」「日常」の3つのテーマ軸**

### 使う言葉（ブランドボキャブラリ）
- 季節：「秋の始まり」「初夏」「冬の温かさ」「新緑」
- 感情：「ご褒美」「寄り添う」「記憶」「ひと息」「やさしさ」「想い」
- 感覚：「香り」「口に含む瞬間」「とろける」「ほのか」「しみじみ」
- 時間：「節目」「人生」「日常」「これから」「あの日」

### 使わない言葉
- 「激安」「最大50%OFF」「今だけ限定」などの煽り文句
- 「美味しい」「おすすめ」の単調な説明
- 過度に甘ったるい・幼稚な表現
- 「テンペラリング」「コンシェージュ」などの専門用語の羅列

## 👥 ターゲット像（30代女性を中心）

### ペルソナ：亜美（35歳、営業職、既婚）
- **購買動機：** 自分へのご褒美、大切な人への贈り物
- **価値観：** 品質・ストーリー・美しさにこだわりがある
- **情報源：** Instagram、クチコミ、ブロガー
- **購買パターン：** 誕生日、記念日、内祝い、手土産
- **心理：** 「ちょっと贅沢でいいもの」に惹かれる。ブランドの背景にある想いを知りたい

### コピーの視点
- 顧客が「何を買うか」より「なぜそれが欲しいのか」を言葉にする
- 「あなたのために選んだ」という丁寧さを表現
- 「自分へのご褒美」と「人へのギフト」の両面性を意識

## 📝 コンテンツ別コピーライティングガイド

### 1️⃣ LP（ランディングページ）

#### ファーストビュー（キャッチコピー）
**構成：** 感情 × 季節 × アクション

**パターン例：**
```
秋の始まりに、ひとかけらの温かさ。
→ 季節（秋）+ 感覚（温かさ）+ 行動（食べる）を一文に

大切な人へ、季節を添える。
→ 関係性（大切な人）+ 季節感 + ギフトニーズ

新しい季節、新しい自分へ。
→ 時間軸の変化 + 「ご褒美」のニュアンス
```

**書き方のコツ：**
- 文字数：15～25字（句読点含む）
- リズム感を大切に（「○○に、○○を」の対比構造）
- 動詞は使わない、名詞と感覚語を組み合わせる

#### サブキャッチ（説明的なキャッチ）
キャッチコピーの次のライン。「なぜ？」に答える。

```
秋の始まりに、ひとかけらの温かさ。
┗ 金木犀の香り、高級ココアの深み。
  季節が変わる瞬間を、一口のお菓子に詰めました。
```

#### CTA（行動喚起）ボタン
**NG：** 「今すぐ購入」「詳しく見る」
**OK：**
- 「季節の彩りを受け取る」
- 「この秋のご褒美を選ぶ」
- 「物語を購入する」

---

### 2️⃣ SNS（Instagram）

#### キャプション基本型
**文字数：** 80～150字（目安）
**構成：** イメージ（1行）→ 背景・素材（2～3行）→ シーン（1行）→ ハッシュタグ（3～5個）

**テンプレート：**
```
[感覚的なオープニング]
秋色に染まったカヌレは、あの日の思い出をそっと甦らせる。

[素材・季節背景]
ボルドーワインとバーベナの香り。
北海道産バターの濃厚さ。
季節の移ろいが、一口に詰まっています。

[使用シーン・感情]
これから始まる季節が、少し好きになる。

[ハッシュタグ]
#oliodo #canelé #季節のお菓子 #秋色 #パティシエ #oliodomoment
```

#### リール動画用キャプション
動きのあるビジュアルに合わせて、テンポよく。

```
秋のカヌレができました。
金木犀を思わせるボルドー色。
バターの香りが、季節を告げる。

今年の秋は、こんなご褒美はいかがですか。

#oliodo #canelé #reels #焼き菓子 #季節限定
```

#### ストーリーズ用キャプション
親しみやすく、カジュアルに。フォロワーとの距離を近める。

```
秋の新作が焼き上がりました🍂
この時期、毎年作る一品。
ボルドーワインの深さ...
バターの香り...
あぁ、秋が来た〜って感じる瞬間。

今週から販売開始です。
```

### 3️⃣ 商品説明（Base ECサイト）

#### 構成（重要：この順序を守る）
1. **第一文：感覚的なオープニング**（口に含んだ瞬間のイメージ）
2. **素材・背景段落**（なぜこの素材？どこから？）
3. **使用シーン段落**（どんな時に、誰と、どう食べるのか）
4. **保存方法など実務情報**

#### テンプレート例：カヌレ

```
【感覚のオープニング】
濃密でしっとり、秋の始まりが香る一口。

【素材・背景】
このカヌレの色は、ボルドーワイン。
選んだ理由は、秋の夕暮れを思い出させたいから。

ボルドーワインと北海道産バターを合わせた生地は、
焼くたびに香りが立ち上ります。
バーベナのほのかな香りが、季節を告げる。

生産者さんから直接仕入れた素材だからこそ、
ここまでこだわることができました。

【使用シーン】
仕事の疲れが出た夜に。
コーヒーと一緒に、ひと息つきたい時に。
大切な人への手土産として。
自分へのご褒美として。

このカヌレは、「季節が変わる瞬間」を美味しく届けたいという想いから生まれました。

【実務情報】
◎ 保存方法：常温で3日間、冷凍で2週間
◎ アレルギー情報：卵、乳、小麦を含む
◎ 内容量：1個（約45g）
```

**ポイント：**
- 「なぜこの素材か」の背景ストーリーを語る
- 「美味しい」ではなく「香りが立ち上ります」など感覚表現
- シーン列挙は「縦書きではなく短い段落」で読みやすく

### 4️⃣ メール・ニュースレター

#### 件名（Subject Line）
**目標：** 「開きたくなる、でも押しつけがましくない」

```
❌ NG例：
【新作キャッシュ50%OFF】今だけ！
秋限定！数量限定！今月のおすすめはコレ！

✅ OK例：
秋のカヌレが焼き上がりました
季節が変わる時、このお菓子のことを思い出してください
あの日の秋の色、覚えていますか？
```

**書き方のコツ：**
- 文字数：25～40字（クライアント側の指定を確認）
- 「限定」「今だけ」ではなく「季節」「この瞬間」で表現
- 相手の「心情」に訴えかける

#### 本文のオープニング
```
こんにちは。

今年も、秋のカヌレが焼き上がりました。

ボルドーワイン、バーベナ、北海道産バター。
毎年この時期、同じ組み合わせで作ります。

なぜなら、「秋はこの色」「秋はこの香り」
という、変わらない想いがあるから。

...
```

**ポイント：**
- 敬語・丁寧さ を保ちながらも「ワタルさんの声」が聞こえる感じ
- 個人的なストーリー感を出す（「毎年この時期」など）
- 「購入してください」ではなく「このお菓子の背景」を伝える

---

### 5️⃣ SNS広告（Facebook・Instagram・Pinterest）

#### テキスト広告（広告クリエイティブ内のテキスト）
**文字数制限：** 最大125字

```
秋の始まりに、ひとかけらの温かさ。

金木犀を思わせるボルドーワイン色のカヌレ。
バターの香りが季節を告げる。

大切な人へのギフトに。
自分へのご褒美に。

今年の秋を彩る、一口のお菓子。
```

**ポイント：**
- 改行を多くして「目に優しい」デザイン
- 最初の一行で「止める」（スクロール止止効果）
- 最後に「行動喚起」（リンク先への興味を高める）

## ✨ 実行上のルール

### 生成時のチェックリスト
- [ ] Oliodoの3つの哲学が反映されているか
- [ ] 「ご褒美」「節目」「日常」いずれかのテーマが含まれているか
- [ ] 季節感（四季の移ろい）が自然に織り交ぜられているか
- [ ] 30代女性の心情に寄り添っているか
- [ ] 「激安」「今だけ」などの煽り文句がないか
- [ ] 「美味しい」の単調な繰り返しがないか

### 生成内容の種類別アプローチ
- **LPコピー** → 「詩的 × 説得力」のバランス を優先
- **SNS投稿** → 「短くても、世界観が伝わる」ことを優先
- **商品説明** → 「ストーリー + 実務情報」の二層構造
- **メール** → 「個人的 + プロフェッショナル」の両立

## 🎓 このスキルを最高品質で使うために

### 事前準備
1. **Oliodo ECサイト（Base）の最新商品** を確認
2. **過去のInstagram投稿** を3～5件確認して、トーン感をキャッチ
3. **ターゲット（30代女性）のニーズ** を具体的にイメージ

### プロンプト例（効果的な指示）
```
「新商品『栗のフィナンシェ』のInstagram投稿キャプションを作成してください。
ターゲットは30代の女性、秋の新作であることを反映してください。
季節感・素材背景・Oliodoのブランド世界観を全て含めてください。」
```

### フィードバック
生成されたコピーがOliodoらしくないと感じたら：
- 「もっと詩的に」「もっと具体的に」など方向性を指示
- 「このコピーのどこが違う？」と具体的にフィードバック
- メモリに「このトーンが好き」と記録して次回に反映
