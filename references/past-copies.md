## 📱 Instagram キャプション例

### ✅ 良い例（Oliodoらしい）

**例1：秋のカヌレ新色発表**
```
秋色に染まったカヌレは、あの日の思い出をそっと甦らせる。

ボルドーワイン、バーベナ、北海道産バター。
毎年この時期、同じ組み合わせで作ります。

なぜなら、秋はこの色だから。
秋はこの香りだから。

#oliodo #canelé #秋色 #季節のお菓子 #oliodomoment
```

**例2：新商品ボンボンショコラ**
```
春の終わりに、桜塩のボンボンショコラが完成しました。

咲いた桜の色、散った桜の香り、
そして少しの塩。

春という季節を、ひとかけらのチョコレートに詰めました。

#oliodo #春限定 #ボンボンショコラ #季節のお菓子
```

**例3：フォロワーとの日常的やり取り**
```
今日、丸一日カヌレを焼きました。

バター、砂糖、卵、小麦粉。
シンプルな素材を、焦らず焼く。

いい色が出てくるまで、待つ。

毎日のこういう時間が好きです。

#oliodo #パティシエ日記 #職人 #焼き菓子 #日常
```

### ❌ NG例（Oliodoらしくない）

**NG1：煽り文句を使ってしまった**
```
秋限定！数量限定！
今だけ50%OFF🎉
この機会を見逃すな🔥
```
→ 「激安」「今だけ」はOliodoのイメージと違う。ブランド毀損。

**NG2：説明的すぎる**
```
カヌレとは、フランスの古典的な焼き菓子で、
カヌレ型の金型を使って焼き上げた、
黒くてカリッとした表面と、
しっとりした内部が特徴です。
```
→ Wikipedia的。顧客の心に届かない。感覚・感情がない。

**NG3：「美味しい」の単調な繰り返し**
```
めっちゃ美味しい！
本当に美味しい！
皆さんもぜひ食べてみてください！
```
→ 幼稚で説得力がない。Oliodoのターゲット（30代女性）には響かない。

**NG4：季節感がない**
```
いつでも食べられる最高のカヌレです。
通年販売なので、いつでも購入できます！
```
→ Oliodo = 「季節の素材」「節目」が大事。季節を無視することはブランド放棄。

## 📄 LPコピー例

### ✅ 良い例

#### ファーストビュー

**例1：秋のキャンペーン**
```
秋の始まりに、
ひとかけらの温かさ。
```

**例2：ギフトキャンペーン（内祝い向け）**
```
人生の彩りを、
一口のお菓子に。
```

**例3：ブランドLP（常設）**
```
いつもの日常に、
一口のご褒美を。
```

#### サブキャッチ＋CTA

```
秋の始まりに、ひとかけらの温かさ。
金木犀の香り、高級ココアの深み。
季節が変わる瞬間を、一口のお菓子に詰めました。

━━━━━━━━━━━━━━━━━━━
季節の彩りを受け取る
━━━━━━━━━━━━━━━━━━━
```

### ❌ NG例

```
秋の新作が激安で登場！最大50%OFF！
売り切れ必至！今だけ限定！
早速購入ボタンをクリック👇
```
→ ディスカウント販売臭が強い。Oliodoの「季節」「節目」の哲学が完全に消えている。

## 🛍️ 商品説明例

### ✅ 良い例

#### カヌレ（秋バージョン）

```
濃密でしっとり、秋の始まりが香る一口。

秋色に染まったカヌレは、ボルドーワイン色。
なぜこの色か？秋の夕暮れを思い出させたいから。

ボルドーワインと北海道産バターを合わせた生地は、
焼くたびに香りが立ち上ります。
バーベナのほのかな香りが、季節を告げる。

生産者さんから直接仕入れた素材だからこそ、
ここまでこだわることができました。

【使用シーン】
仕事の疲れが出た夜に。
コーヒーと一緒に、ひと息つきたい時に。
大切な人への手土産として。
自分へのご褒美として。

このカヌレは、「季節が変わる瞬間」を美味しく届けたいという想いから生まれました。

【商品情報】
内容量：1個（約45g）
保存方法：常温で3日間、冷凍で2週間
アレルギー情報：卵、乳、小麦を含む
```

### ❌ NG例

```
当店の秋限定カヌレ。
フランスの伝統的な焼き菓子をアレンジしました。
黒くてカリッとした外側と、しっとりした内部が特徴。
ボルドーワイン色で、見た目も美しい。
高級ココアをたっぷり使用しているので、とっても美味しいです。
```
→ 説明的で、顧客の感情に訴えかけていない。
→ 「美味しい」の単調な繰り返し。
→ 「なぜボルドーワイン色なのか」という背景ストーリーがない。

## 📊 対比表：OK vs NG

| 要素 | OK（Oliodoらしい） | NG（ブランド毀損） |
|---|---|---|
| **ファーストビュー** | 秋の始まりに、ひとかけらの温かさ | 秋限定！今だけ50%OFF |
| **季節表現** | 「金木犀を思わせる」「秋色」 | 「季節関係なく通年販売」 |
| **商品説明** | 「口に含むと、バターの香りが...」 | 「美味しいです！」 |
| **感情軸** | 「ご褒美」「節目」「記憶」 | 「お得」「激安」「限定」 |
| **ターゲット言語** | 詩的、上品、大人っぽい | カジュアル、幼稚、煽り気味 |
| **CTA文** | 「季節の彩りを受け取る」 | 「今すぐ購入」「買う」 |
| **ブランドボイス** | ワタルさんの個人的な想い | 誰にでも当てはまる説明 |

## 🎯 ポイント（覚えておくこと）

1. **「説明」ではなく「物語」を語る**
   - 「カヌレは〇〇という菓子です」ではなく
   - 「秋という季節が、この色とこの香りで甦る」

2. **顧客の「心情」に寄り添う**
   - 「買ってください」ではなく
   - 「この秋、あなたはどう過ごしたいですか？」

3. **Oliodoの3つの哲学を常に問い直す**
   - 季節感は？→ 含まれているか
   - 職人の想いは？→ 伝わっているか
   - 感謝は？→ 生産者や顧客への想いは見えるか

4. **30代女性の「こっそり贅沢」欲を刺激**
   - 「自分へのご褒美」「大切な人へのギフト」という二面性
   - 高級感は欲しいが、敷居は高くない
