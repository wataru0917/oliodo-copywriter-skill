# Oliodo Copywriter Skill

Oliodo（オリオド）の **全マーケティングコンテンツ** で一貫性のある、感情的で洗練されたコピーを生成するスキル。

## 概要

このスキルは、焼き菓子・洋菓子ブランド「Oliodo」の以下のコンテンツを対象としています：

- 🖇️ ランディングページ（LP）のキャッチコピー・見出し・CTA
- 📱 Instagram投稿のキャプション・ハッシュタグ
- ✉️ メールマガジン・ニュースレター
- 🛍️ Base ECサイトの商品説明
- 📢 SNS広告（Facebook・Instagram・Pinterest）
- 📰 プレスリリース・ブランドメッセージング

**特徴：**
- Oliodoの3つの哲学（季節感・職人性・感謝）を常に反映
- 30代女性のハート（「ご褒美」「節目」「日常」）に寄り添う
- 詩的で上品、でも押しつけがましくない  
- 「激安」「今だけ」などの煽り文句を一切使わない

## 🎯 Oliodoの3つの哲学（常に確認）

このスキルを使うたびに、以下を問い直してください：

### 1. 季節の素材の活用
- コピーに季節イメージが自然に織り交ぜられているか
- 「秋」「春」「初夏」など時間軸が見えるか

### 2. クリエイター・職人としての探究
- 「なぜこの素材なのか」の背景が語られているか
- 職人目線より「顧客がいかに感動するか」か

### 3. 関係者への感謝
- 生産者・顧客・仲間へのリスペクトが見えるか
- 「作り手の想い」が顧客に伝わっているか

## ✨ コンテンツ別のポイント

### LPコピー
- **最優先：** 「詩的 × 説得力」のバランス
- キャッチコピー 15～25字、対比構造（「○○に、○○を」）
- NG：「美味しい」の単調な繰り返し

### SNS投稿（Instagram）
- **最優先：** 「短くても、世界観が伝わる」
- 文字数 80～150字
- イメージ → 背景 → シーン → ハッシュタグの構成

### 商品説明
- **最優先：** 「ストーリー + 実務情報」の二層構造
- 感覚的なオープニング → 素材・背景 → 使用シーン → 実務情報
- NG：「説明的」「Wikipedia的」

### メール件名
- **最優先：** 「開きたくなる、でも押しつけがましくない」
- 文字数 25～40字
- NG：「今だけ50%OFF」「数量限定」

### SNS広告
- **最優先：** 「感情訴求 + 行動喚起」
- 文字数 125字以内
- スクロール止止効果 → 最後に明確なCTA

## 📊 テストの実行方法

### マニュアルテスト

`tests/test_cases.json` の各テストケースを実行：

```
例1：LP カヌレ キャッチコピー
├─ prompt: "秋限定カヌレのLPファーストビュー用キャッチコピーを..."
├─ expected_characteristics: [季節感、感情訴求、詩的表現...]
└─ 生成 → 評価

例2：Instagram 栗フィナンシェ
├─ prompt: "新商品『栗のフィナンシェ』のInstagram投稿キャプション..."
└─ 生成 → 評価
```

### 評価基準

各テストケースには以下の評価軸があります：

1. **ブランド一貫性（40%）** — Oliodo哲学が反映されているか
2. **感情訴求度（30%）** — 30代女性のハートに寄り添っているか
3. **技術品質（30%）** — コピー技法（文字数・構成・言葉選び）が正確か

**合格基準：** 品質基準の70%以上が「YES」で、かつ「Oliodo Philosophy」が含まれていること

## 💡 ベストプラクティス

### DO（やるべき）
- ✅ 各コピー生成前に、`SKILL.md` と `references/past-copies.md` を参照する
- ✅ テストケースを定期的に実行して、スキルの精度を確認する
- ✅ 新しいコピー例が完成したら、`past-copies.md` に追加して、チームで共有する
- ✅ GitHub 上でVersion管理を行い、過去のコピー案も残しておく

### DON'T（避けるべき）
- ❌ 「激安」「今だけ」などの煽り文句を使う
- ❌ 「美味しい」「素晴らしい」の単調な繰り返し
- ❌ ターゲット（30代女性）視点を忘れ、オタクな説明になる
- ❌ 季節感を無視する（Oliodo = 「季節」が生命線）
- ❌ GitHub を更新せず、ローカルだけで完結させる

### Q: 生成されたコピーがOliodoらしくない
